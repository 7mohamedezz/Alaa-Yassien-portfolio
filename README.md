# Alaa Yassien Portfolio

Personal portfolio website for Alaa Yassien, showcasing projects, skills, and contact information.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui + Radix UI

## Getting Started

### Prerequisites

- Node.js (recommended: latest LTS)
- npm (or Bun)

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Lint

```bash
npm run lint
```

## Tests

```bash
npm run test
```

## Project Structure

- `src/` - application source
- `public/` - static assets

## Deployment

This project can be deployed on any static hosting provider (e.g., Vercel, Netlify).
